üîß Langkah-Langkah Setup yang Telah Dilakukan

1Analisis Proyek

Mengidentifikasi bahwa proyek ini adalah aplikasi CLI berbasis Laravel Zero dengan database SQLite.

  • Framework: Laravel Zero v12.0.2
  • Memerlukan PHP 8.2 atau lebih tinggi
  • Menggunakan Composer untuk dependency management
  • Database: SQLite (file-based, tidak perlu server database)

2Menemukan PHP dari Laragon

Ditemukan bahwa sistem Anda sudah memiliki Laragon terinstall dengan PHP di dalamnya.

  • ‚úì PHP 8.3.26 ditemukan di: C:\laragon\bin\php\php-8.3.26-Win32-vs16-x64
  • ‚úì Versi PHP memenuhi requirement (minimal 8.2)
  • ‚úì Composer 2.8.12 sudah terinstall

3Menambahkan PHP ke PATH Windows

Agar PHP bisa diakses dari PowerShell, ditambahkan ke environment variable PATH.

$phpPath = "C:\laragon\bin\php\php-8.3.26-Win32-vs16-x64"
[Environment]::SetEnvironmentVariable("Path", $env:Path + ";$phpPath", "User")
Hasil: PHP berhasil ditambahkan ke PATH dan dapat diakses dari terminal.

4Mengaktifkan Ekstensi ZIP di PHP

Ekstensi ZIP diperlukan agar Composer bisa extract packages.

  • File konfigurasi: C:\laragon\bin\php\php-8.3.26-Win32-vs16-x64\php.ini
  • Mengubah ;extension=zip menjadi extension=zip
Hasil: Ekstensi ZIP berhasil diaktifkan.

5Install Dependencies dengan Composer

Menginstall semua package PHP yang diperlukan oleh aplikasi.

composer install --no-interaction --ignore-platform-req=ext-posix
Catatan Penting: Flag --ignore-platform-req=ext-posix diperlukan karena:
  • Ekstensi posix hanya tersedia di Linux/Unix
  • Package php-school/cli-menu memerlukan ekstensi ini
  • Aplikasi tetap berfungsi di Windows tanpa ekstensi ini (untuk perintah non-interaktif)
Hasil: 113 packages berhasil terinstall.

6Setup Database SQLite

Membuat file database SQLite dan menjalankan migrasi untuk membuat tabel-tabel.

Membuat File Database:

New-Item -ItemType File -Path "database\database.sqlite" -Force

Menjalankan Migrasi:

php sherina-mart migrate --force
Hasil Migrasi:
  • ‚úì categories - Tabel kategori produk (6.92ms)
  • ‚úì varieties - Tabel variasi produk (4.88ms)
  • ‚úì products - Tabel data produk (5.12ms)
  • ‚úì sale_transactions - Tabel transaksi penjualan (4.55ms)
Ukuran Database: 45,056 bytes (45 KB)

üöÄ Cara Menjalankan Aplikasi

Metode 1: PowerShell (Untuk Perintah Non-Interaktif)

# Refresh environment variables
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")

# Navigasi ke folder proyek
cd d:\project\sherina-mart-main\sherina-mart-main

# Jalankan perintah
php sherina-mart list           # Lihat semua perintah
php sherina-mart migrate:status # Status migrasi
php sherina-mart inspire        # Quote inspiratif
php sherina-mart db:seed        # Seed database

Metode 2: WSL (Untuk Menu Interaktif)

Direkomendasikan untuk menu interaktif 
# Install WSL (jalankan sebagai Administrator)
wsl --install

# Restart komputer

# Setelah restart, install PHP di WSL
sudo apt update
sudo apt install php php-cli php-sqlite3 php-mbstring composer

# Jalankan aplikasi
cd /mnt/d/project/sherina-mart-main/sherina-mart-main
php sherina-mart app:menu-command

Metode 3: Git Bash (Alternatif untuk Menu Interaktif)

  1. Download Git for Windows dari: https://git-scm.com/download/win
  2. Install Git Bash
  3. Buka Git Bash dan jalankan: 
    cd /d/project/sherina-mart-main/sherina-mart-main
php sherina-mart app:menu-command

‚ö†Ô∏è Masalah yang Ditemukan dan Solusinya

Masalah 1: PHP tidak terdeteksi

Error: php : The term 'php' is not recognized
Solusi: Refresh environment variables atau restart PowerShell: 
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")

Masalah 2: Ekstensi ZIP tidak aktif

Error: The zip extension and unzip/7z commands are both missing
Solusi: Aktifkan ekstensi ZIP di php.ini dengan mengubah ;extension=zip menjadi extension=zip

Masalah 3: Ekstensi POSIX diperlukan

Error: ext-posix * -> it is missing from your system
Solusi: Gunakan flag --ignore-platform-req=ext-posix saat install Composer

Masalah 4: Menu interaktif tidak berfungsi

Error: 'stty' is not recognized, 'tput' is not recognized
Penjelasan: Library php-school/cli-menu memerlukan perintah Linux (stty, tput) yang tidak tersedia di Windows PowerShell.
Solusi:
  • Opsi 1: Gunakan WSL (Windows Subsystem for Linux)
  • Opsi 2: Gunakan Git Bash
  • Opsi 3: Gunakan perintah non-interaktif saja
